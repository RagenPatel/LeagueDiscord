# LeagueDiscord

Bot that monitors Discord chatroom and replies to questions by going to various APIs to retrieve and manipulate data.

# Setup

1. Set up environment variables for discord API token, Riot games API, and ChampionGG API in ~/.bash_profile file with `export DISCORD_API_TOKEN=*TOKEN HERE*` .. use `RIOT_API` and `CHAMP_API` for the other two keys.
2. Install required dependencies by running the dependencies.sh file
